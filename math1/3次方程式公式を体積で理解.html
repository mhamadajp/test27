$(a+b)^3$ を直方体の「体積の足し算」として視覚的に理解しよう！

(a + b)³ = a³ + 3a²b + 3ab² + b³

マウスの**左ドラッグで回転**、**スクロールでズーム**、**右ドラッグで移動**ができます。
いろいろな角度からブロックを観察してみてください。

💡

なぜ 3a²b や 3ab² なの？

大きな立方体（一辺が a+b）を分解すると、実は**8つの直方体**の集まりになっています。

  • a³ : 一辺 a の立方体が 1個
  • b³ : 一辺 b の立方体が 1個
  • a²b : a×a×b の板状ブロックが 3個
  • ab² : a×b×b の棒状ブロックが 3個
